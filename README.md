# TeleTurtle

Le jeu à succès TeleTurtle
Contrôle une tortue vivant une vie séparé dans 2 carapaces.
Résous des énigmes pour atteindre les légumes.

Touches:
 ZQSD pour bouger, avec la souris pour orienté la caméra
 G pour passer d'une carapace à l'autre
 Clique droit pour ramasser l'autre carapace
 Clique gauche pour lancer la carapace (rester appuyer pour augmenter la force du lancé)
